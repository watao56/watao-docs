# DeadLink - リンク切れ自動検知サービス

## 概要

ブログ・Webサイトのリンク切れを定期的に検知し、通知するSaaSサービス。

URLを入力するだけで即座にリンク切れをチェックでき、登録不要・無料で利用可能。定期監視機能により、SEO低下を未然に防ぎます。

## ターゲットユーザー

- **メイン**: 記事50本以上の中堅〜ベテランブロガー・技術ブロガー
- **サブ**: 小規模メディア運営者、ポートフォリオサイト管理者

## 主要機能

1. **即スキャン** - 登録不要でURLを入力するだけ（30秒以内に結果表示）
2. **定期監視** - 自動でリンク切れをチェック
3. **通知** - メール / Discord / Slack で即通知
4. **ダッシュボード** - サイト管理・結果一覧
5. **シェア可能な結果ページ** - スキャン結果をURLで共有
