# OpenAI Codex CLI

## 概要

**OpenAI Codex CLI**は、OpenAIが開発したオープンソースのコーディングエージェントです。ターミナル上で直接動作し、リポジトリの調査、ファイル編集、コマンド実行を自動化できる軽量なツールです。

!!! tip "最新バージョン"
    現在の最新版は**v0.94.0**（2026年2月）

## 特徴と強み

### 🚀 ローカル実行
- **完全にローカル環境で動作**：コードが外部に送信されることなく、セキュリティを重視
- **サンドボックス環境対応**：安全な環境でコード実行とテストが可能
- **オフライン編集モード**：一部機能はオフラインでも利用可能

### 🔧 マルチプロバイダ対応
- **OpenAI GPT-4**：標準搭載（GPT-4, GPT-4 Turbo）
- **他プロバイダ対応**：Anthropic Claude、Google Gemini等も設定可能
- **カスタムエンドポイント**：企業内LLMや自前モデルとの統合

### ⚡ 軽量＆高速
- **最小限の依存関係**：Node.js環境があれば即座に実行可能
- **効率的なdiff生成**：必要最小限の変更のみを提案
- **バッチ処理**：複数ファイルの一括編集をサポート

### 🛠️ 豊富な自動化モード
- **auto-edit**：指定したファイルを自動的に編集
- **full-auto**：プロジェクト全体を分析し、関連ファイルも含めて包括的に修正
- **インタラクティブモード**：ユーザーとの対話を通じて段階的に作業

## Claude Code / Gemini CLI との比較

| 項目 | **Codex CLI** | **Claude Code** | **Gemini CLI** |
|------|---------------|-----------------|----------------|
| **料金** | 🟢 リーズナブル | 🟡 やや高額 | 🟢 安価 |
| **編集精度** | 🟢 高精度・最小diff | 🟡 やや冗長 | 🟡 標準的 |
| **MCP対応** | 🟡 stdio経由のみ | 🟢 ネイティブサポート | 🟡 限定的 |
| **コンテキスト容量** | 🟡 標準的 | 🟢 大容量 | 🟢 大容量 |
| **UI/UX** | 🟡 シンプル | 🟢 直感的 | 🟡 標準的 |
| **安定性** | 🟢 高い信頼性 | 🟡 時々過度な修正 | 🟡 標準的 |

### いつCodex CLIを選ぶべきか？

**Codex CLIが最適な場面：**
- 💰 **コストを抑えたい**プロジェクト
- 🎯 **正確で最小限の変更**を重視する場合
- ⚡ **軽量で高速**な動作が必要
- 🔒 **セキュリティ**を最優先にしたい企業環境

**他ツールが適している場面：**
- **Claude Code**: 複雑な説明・計画・大規模リファクタリング
- **Gemini CLI**: 実験的な機能やコスト最優先の場面
