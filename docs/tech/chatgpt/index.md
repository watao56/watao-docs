# ChatGPT 包括ガイド

OpenAIが提供するChatGPTは、現在最も広く使われている生成AIサービスの一つです。2026年2月現在、豊富な機能と多様なモデルによって、個人利用から企業利用まで幅広く活用されています。

!!! tip "最新情報"
    2025年12月にGPT-5.2がリリースされ、さらに2026年1月にはChatGPT Goプラン（月額$8）も新登場しました。また、o3-proやo4-miniなどの高性能推論モデルも追加されています。

## ChatGPTとは

ChatGPTは、OpenAIが開発した対話型の大規模言語モデル（LLM）です。自然な会話形式でテキスト生成、コード作成、分析、翻訳など多岐にわたるタスクを実行できます。

### 主要な特徴

- **多モーダル対応**: テキスト、画像、音声、動画に対応
- **リアルタイム性**: 最新情報の検索とブラウジング機能
- **カスタマイズ性**: カスタム指示とカスタムGPTsによる個人化
- **統合機能**: Canvas、DALL-E画像生成、Advanced Data Analysisなど

## プラン別機能比較

### 無料プラン
- **利用可能モデル**: GPT-5.2、GPT-4o（制限あり）
- **制限**: 時間制限、メッセージ数制限
- **機能**: 基本的な対話、制限付きの機能

!!! warning "注意"
    無料プランではThinkingモード、長文コンテキスト、o系推論モデルは利用できません。

### ChatGPT Go（月額$8・1,500円）
- **利用可能モデル**: GPT-5.2 Instantのみ
- **特徴**: 無料版より多く利用可能、初心者向け
- **制限**: 上位モデル（o3、o4など）は利用不可

### ChatGPT Plus（月額$20・約3,000円）
- **利用可能モデル**: GPT-4o無制限、o3、o4-mini、o4-mini-high
- **利用制限**:
  - GPT-4o: 3時間ごとに80メッセージ
  - o3: 週50メッセージ
  - o4-mini: 日150メッセージ
  - o4-mini-high: 日50メッセージ
- **プレミアム機能**: Canvas、DALL-E画像生成、Web検索、カスタムGPTs

### ChatGPT Pro（月額$200・約30,000円）
- **対象**: 研究者、開発者、プロフェッショナル
- **特徴**: o3-proへのアクセス、より高い利用制限
- **追加機能**: APIアクセス、高度な分析機能

### ChatGPT Team（月額$25/ユーザー）
- **対象**: チーム・小規模企業
- **機能**: 共有ワークスペース、管理機能、カスタムGPT共有

### ChatGPT Enterprise/Edu
- **対象**: 大企業・教育機関
- **特徴**: エンタープライズレベルのセキュリティ、カスタマイゼーション

## 利用可能なモデル一覧

### GPTシリーズ

#### GPT-5.2（2025年12月リリース）
- **特徴**: OpenAI最高性能モデル
- **得意分野**: 複雑な推論、創作、高度な分析
- **利用**: 無料〜Proプランすべて

#### GPT-4o
- **特徴**: マルチモーダル対応、高速応答
- **得意分野**: 画像認識、コード生成、一般的なタスク
- **利用**: Plus以上で無制限

### o系推論モデル

#### o3（2025年6月リリース）
- **特徴**: 高度な推論能力、思考プロセスの可視化
- **得意分野**: 数学、論理問題、複雑な分析
- **利用**: Plus（週50）、Pro（より多く）

#### o3-pro（2025年6月リリース）
- **特徴**: o3の最上位版、最長思考時間
- **得意分野**: 研究レベルの高度な問題解決
- **利用**: Proプラン限定

#### o4-mini（2025年4月リリース）
- **特徴**: o3の軽量版、高速・低コスト
- **得意分野**: 日常的な推論タスク
- **利用**: Plus（日150）、Pro（より多く）

#### o4-mini-high
- **特徴**: o4-miniの高精度版
- **得意分野**: より正確性が必要なタスク
- **利用**: Plus（日50）、Pro（より多く）

## 主要機能の概要

### Canvas
リアルタイム編集が可能な共同作業機能。文章執筆やコーディングで威力を発揮します。

!!! example "Canvas活用例"
    - 長文記事の執筆と編集
    - HTMLアプリの開発とプレビュー
    - バージョン管理付きの共同編集

### 画像生成（DALL-E → GPT Image）
2025年にDALL-E 3からGPT Imageに移行し、より高品質な画像生成が可能になりました。

### Advanced Data Analysis（旧Code Interpreter）
- Pythonコード実行
- データ分析・可視化
- ファイルアップロード・処理

### Web検索・Browse機能
リアルタイムの情報検索と最新データの取得が可能です。

### Voice Mode
- 音声での対話
- 標準モードと高度モードを選択可能
- 自然な会話フロー

### メモリ機能
- 過去の会話履歴の記憶
- 個人的な情報や設定の保存
- 継続的な文脈理解

### カスタムGPTs
特定用途に特化したGPTを作成・利用可能：

- **開発**: コードレビュー、デバッグ支援
- **文章**: ライティング支援、校正
- **分析**: データ分析、レポート生成
- **教育**: 学習支援、問題作成

### Record機能（macOSのみ）
- 音声・動画の録画と文字起こし
- 自動要約とアクションアイテム抽出
- Pro、Enterprise、Edu向け

## 利用シーン別推奨プラン

| 用途 | 推奨プラン | 理由 |
|------|------------|------|
| 軽い文章作成・質問 | 無料/Go | 基本機能で十分 |
| 本格的なライティング | Plus | Canvas、無制限利用 |
| プログラミング | Plus/Pro | o系モデル、ADA利用 |
| 研究・分析 | Pro | o3-pro、高利用制限 |
| チーム利用 | Team | 共有機能、管理機能 |
| 企業利用 | Enterprise | セキュリティ、カスタマイズ |

## まとめ

ChatGPTは2026年現在、最も多機能で実用的な生成AIサービスです。無料プランでも十分な機能を提供しながら、有料プランでは高度な推論モデルやプレミアム機能により、プロフェッショナルな用途にも対応できます。

特に以下の点で他のAIサービスと差別化されています：

- **豊富なモデル選択肢**: 用途に応じた最適なモデル
- **統合された機能**: Canvas、画像生成、データ分析等
- **カスタマイズ性**: カスタム指示とカスタムGPTs
- **幅広い価格帯**: 無料から企業向けまで

自分の利用目的と予算に応じて適切なプランを選択し、効果的なプロンプト設計と組み合わせることで、ChatGPTの真価を発揮できるでしょう。
