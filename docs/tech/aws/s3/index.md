# Amazon S3 - Simple Storage Service

!!! info "このページについて"
    Amazon S3の概要とストレージクラスについて解説します。実際の設定手順は[セットアップガイド](setup.md)を、活用事例は[ユースケース](use-cases.md)をご覧ください。

## S3とは

Amazon Simple Storage Service（S3）は、Webアプリケーション、モバイルアプリケーション、エンタープライズアプリケーション、IoTデバイスなど、どこからでもデータを格納・取得できる**オブジェクトストレージサービス**です。

### 特徴

- **高い耐久性**: 99.999999999%（イレブンナイン）の耐久性
- **無制限のストレージ**: 容量制限なし
- **グローバル**: 世界中のリージョンで利用可能
- **多様なストレージクラス**: コストと用途に応じて選択可能

!!! tip "オブジェクトストレージとは"
    従来のファイルシステムとは異なり、フラットな構造でファイル（オブジェクト）を保存する方式です。各オブジェクトにはユニークなキー（パス）が付与され、URLでアクセス可能です。

## ストレージクラス

S3では、データのアクセス頻度と取得時間要件に基づいて複数のストレージクラスを提供しています。

### 1. S3 Standard

| 項目 | 詳細 |
|------|------|
| 用途 | 頻繁にアクセスするデータ |
| 可用性 | 99.99% |
| 最小保存期間 | なし |
| 取り出し料金 | なし |
| 初回バイト待ち時間 | ミリ秒 |

**使用例**: ウェブサイト画像、動的コンテンツ、頻繁にアクセスされるファイル

### 2. S3 Standard-Infrequent Access（IA）

| 項目 | 詳細 |
|------|------|
| 用途 | 月に1回程度アクセスするデータ |
| 可用性 | 99.9% |
| 最小保存期間 | 30日 |
| 取り出し料金 | あり |
| 初回バイト待ち時間 | ミリ秒 |

**使用例**: バックアップファイル、災害復旧データ

!!! warning "注意点"
    30日未満で削除すると、30日分の料金が請求されます。

### 3. S3 Glacier Instant Retrieval

| 項目 | 詳細 |
|------|------|
| 用途 | 四半期に1回程度アクセスするデータ |
| 可用性 | 99.9% |
| 最小保存期間 | 90日 |
| 取り出し料金 | あり |
| 初回バイト待ち時間 | ミリ秒 |

**使用例**: アーカイブデータ（即座にアクセス必要）

### 4. S3 Glacier Flexible Retrieval

| 項目 | 詳細 |
|------|------|
| 用途 | 年に1-2回程度アクセスするデータ |
| 可用性 | 99.99% |
| 最小保存期間 | 90日 |
| 取り出し料金 | あり |
| 初回バイト待ち時間 | 分〜時間 |

**取得オプション**:
- **緊急**: 1-5分（高コスト）
- **標準**: 3-5時間（中コスト）
- **一括**: 5-12時間（低コスト）

### 5. S3 Glacier Deep Archive

| 項目 | 詳細 |
|------|------|
| 用途 | 7-10年以上の長期保存データ |
| 可用性 | 99.99% |
| 最小保存期間 | 180日 |
| 取り出し料金 | あり |
| 初回バイト待ち時間 | 12時間〜48時間 |

**取得オプション**:
- **標準**: 12時間
- **一括**: 48時間（低コスト）

### 6. S3 Intelligent-Tiering

機械学習によるアクセスパターン分析で、自動的に最適なストレージクラスに移動します。

!!! tip "Intelligent-Tieringの階層"
    - **高頻度階層**: Standard相当
    - **低頻度階層**: IA相当
    - **アーカイブインスタント階層**: Glacier Instant Retrieval相当
    - **アーカイブアクセス階層**: Glacier Flexible Retrieval相当
    - **ディープアーカイブ階層**: Glacier Deep Archive相当

## 料金体系

### 2025年現在の東京リージョン料金（目安）

| ストレージクラス | ストレージ料金（GB/月） | 取り出し料金（GB） |
|------------------|------------------------|-------------------|
| Standard | $0.025 | 無料 |
| Standard-IA | $0.019 | $0.01 |
| Glacier Instant | $0.007 | $0.03 |
| Glacier Flexible | $0.005 | $0.01-0.05 |
| Glacier Deep Archive | $0.002 | $0.02 |
| Intelligent-Tiering | $0.025 + 監視料金 | 無料（階層移動時） |

!!! warning "料金の注意点"
    - 上記は目安価格です。最新の料金は[AWS公式ページ](https://aws.amazon.com/s3/pricing/)をご確認ください
    - リクエスト料金（PUT、GET等）が別途発生します
    - データ転送料金（特にCloudFront以外への転送）も考慮が必要です

### その他の料金要素

- **リクエスト料金**: PUT、POST、GET等の操作ごと
- **データ転送料金**: インターネットへの転送
- **管理・分析料金**: S3 Analytics、Inventory等

## ユースケース概要

### 1. ウェブアプリケーション
- 静的コンテンツ配信
- ユーザーアップロードファイル保存
- ログファイル保存

### 2. バックアップ・アーカイブ
- データベースバックアップ
- 法的要件による長期保存
- 災害復旧用データ

### 3. データ分析
- データレイク構築
- ビッグデータ分析基盤
- 機械学習データセット

### 4. コンテンツ配信
- メディアファイル（画像、動画、音声）
- ソフトウェア配布
- CDN（CloudFront）との組み合わせ
