# 💜 Claude — 包括ガイド

> **Claude（クロード）** は Anthropic が開発した次世代対話型 AI アシスタントです。
> 2026年現在、最上位モデル Claude Opus 4.6 を含む豊富なモデルラインナップで、あらゆるタスクに対応できます。

## Claudeとは

**Claude** は Anthropic が開発した大規模言語モデル（LLM）ベースの AI アシスタントです。安全性と有用性のバランスを重視した設計で、自然な対話から高度なコーディング、複雑な推論まで幅広いタスクを処理できます。

### 💡 Claude の特徴

- **自然な日本語対応**: 高品質で自然な日本語での対話
- **200K〜1M トークンコンテキスト**: 長文資料の処理が得意
- **XML タグ活用**: Claude 独自の強力なプロンプトテクニック
- **安全性重視**: Constitutional AI による安全で信頼できる応答
- **豊富な機能**: Artifacts、Projects、Computer Use など多彩な機能

## プラン比較

### 🆓 Free プラン

- **料金**: 無料
- **利用制限**: 日/週単位でメッセージ数制限
- **対象モデル**: Sonnet 4.5、Haiku 4.5
- **機能**: 基本対話、Web検索、ファイルアップロード（制限あり）

### 💰 Pro プラン（$20/月）

- **料金**: 月額 $20
- **利用制限**: 大幅に緩和（Opus 4.6 も利用可能）
- **機能**: 全モデルアクセス、Projects、Advanced Reasoning
- **追加**: Extended Thinking モード

### 🌟 Max プラン（$35/月）

- **料金**: 月額 $35
- **特徴**: Pro プランの全機能 + より多い利用枠
- **推奨**: ヘビーユーザー向け

### 👥 Team プラン（$30/月/ユーザー）

- **料金**: 月額 $30/ユーザー（5ユーザー〜）
- **機能**: Pro 機能 + チーム共有、管理機能
- **追加**: Project 共有、使用状況分析

### 🏢 Enterprise プラン

- **料金**: 要相談
- **機能**: Team 機能 + SSO、高度なセキュリティ、専用サポート
- **特徴**: データレジデンシー制御、カスタム統合

!!! tip "プラン選択指針"
    - **個人学習**: Free → Pro
    - **プロ利用**: Max
    - **チーム開発**: Team
    - **企業利用**: Enterprise

## 主要機能概要

### ✨ Artifacts

**Artifacts（アーティファクト）** は Claude の代表的機能で、対話の中でコード実行、文書作成、データ可視化などを行えます。

**できること**:
- **コード実行**: Python、JavaScript、HTML/CSS
- **データ分析**: CSV データの可視化・分析
- **文書作成**: 構造化された文書の生成・編集
- **UI作成**: インタラクティブなWebアプリ

[詳細: モード・機能活用](modes.md#artifacts)

### 📁 Projects

**Projects（プロジェクト）** は特定タスク向けにカスタマイズされた Claude 環境です。

**特徴**:
- **カスタム指示**: タスク特化の動作設定
- **Knowledge**: 関連資料のアップロード・参照
- **一貫性**: 同じ設定での継続作業

ChatGPT の「カスタム GPT」、Gemini の「Gems」に相当する機能です。

[詳細: Projects・カスタム指示](projects.md)

### 🧠 Extended Thinking

**Extended Thinking** は Claude 4シリーズの新機能で、問題を段階的に深く考察する推論モードです。

**特徴**:
- より深い推論プロセスの可視化
- 複雑な問題の段階的解決
- 思考過程の透明性

### 🔗 MCP（Model Context Protocol）

**MCP** は Claude が外部システムと連携するためのプロトコルです。

**活用例**:
- ファイルシステムアクセス
- データベース操作
- API 統合
- カスタムツール連携

[詳細: エコシステム](ecosystem.md#mcp)

### 👁️ Vision

画像解析機能で、以下が可能です：

- 画像内容の詳細説明
- OCR（文字認識）
- チャート・グラフ分析
- 画像ベースのコーディング支援

### 🖱️ Computer Use

実際のコンピューター操作を AI が代行する革新的機能です。

**できること**:
- 画面の視覚的認識
- マウス・キーボード操作
- ブラウザ操作
- アプリケーション制御
