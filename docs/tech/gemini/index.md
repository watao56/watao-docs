# Gemini CLI 概要

## Gemini CLIとは

[Gemini CLI](https://github.com/google-gemini/gemini-cli) は、Google が開発したオープンソースのAIコーディングエージェントです。ターミナル上で直接 Gemini モデルを活用でき、コード生成・編集・デバッグ・ファイル操作などを自然言語で指示できます。

Claude Code と同じカテゴリの「ターミナルベース AI コーディングエージェント」であり、Apache 2.0 ライセンスで公開されています。

```bash
# インストール不要で即座に試せる
npx @google/gemini-cli
```

## 特徴と強み

### 🎯 無料枠が非常に充実

| 項目 | 無料枠 |
|------|--------|
| リクエスト数 | 60回/分、1,000回/日 |
| モデル | Gemini 3（Pro/Flash混合） |
| コンテキストウィンドウ | 100万トークン |
| 認証 | Google アカウントでログインするだけ |

!!! tip "個人開発なら無料で十分"
    1日1,000リクエストは、個人の開発作業では十分な量です。API キーの管理も不要で、Google アカウントがあればすぐに始められます。

### 🧠 大規模コンテキストウィンドウ

Gemini 3 では**100万トークン**のコンテキストウィンドウを持ち、大規模なコードベース全体を一度に読み込んで分析できます。

- プロジェクト全体のアーキテクチャ理解
- 大規模リファクタリングの計画
- コードベース横断的なバグ調査

### 🔧 充実のビルトインツール

- **Google Search グラウンディング**: リアルタイム情報で回答を根拠づけ
- **ファイル操作**: コードの読み取り・編集・作成
- **シェルコマンド実行**: ターミナルコマンドの実行（確認付き）
- **Web フェッチ**: URL からの情報取得
- **マルチモーダル**: 画像・PDF・動画からのコード生成

### 🔌 MCP（Model Context Protocol）対応

外部ツールやサービスとの連携を MCP サーバー経由で拡張可能。データベース、API、カスタムスクリプトなど、あらゆる外部システムと接続できます。

### 🤖 GitHub Actions 統合

[Gemini CLI GitHub Action](https://github.com/google-github-actions/run-gemini-cli) を使って、PR レビュー、Issue トリアージ、`@gemini-cli` メンション対応などを自動化できます。

## Claude Code との比較

| 項目 | Gemini CLI | Claude Code |
|------|-----------|-------------|
| **開発元** | Google | Anthropic |
| **基盤モデル** | Gemini 3 Pro / 2.5 Pro | Claude Opus 4 / Sonnet 4 |
| **無料枠** | ✅ 1,000回/日 | ❌ Pro プラン $20/月〜 |
| **コンテキスト** | 100万トークン | 100万トークン（Sonnet 4） |
| **ライセンス** | Apache 2.0（オープンソース） | プロプライエタリ |
| **MCP 対応** | ✅ | ✅ |
| **Google Search** | ✅ ビルトイン | ❌ |
| **マルチモーダル** | ✅ 画像・PDF・動画 | ✅ 画像 |
| **コード品質** | 良好（大規模分析に強い） | 非常に高い（精密な実装に強い） |
| **推論能力** | 良好 | 非常に高い（複雑なロジックに強い） |

### 使い分けの指針

!!! note "基本方針"
    - **Gemini CLI**: 大規模な分析・調査、無料で気軽に使いたい場面、マルチモーダル入力
    - **Claude Code**: 精密なコード実装、複雑なロジック、高品質なリファクタリング

詳しい連携パターンは [Claude Codeとの連携](integration.md) を参照してください。
