# モード活用ガイド

Geminiアプリには様々なモデルと専門機能があります。それぞれの特徴を理解し、タスクに最適な機能を選択することで、効率的かつ高品質な結果を得られます。

## モデル選択と使い分け

### Gemini 3 Pro（最高性能）

**適用場面**:
- 複雑な論理的思考が必要なタスク
- 高度な創作活動
- 専門的な分析・研究
- 長文の構造化された文書作成

**具体例**:
```
✅ 法的文書の分析
✅ 学術論文の執筆支援
✅ 複雑なビジネス戦略立案
✅ 高度なプログラミング問題
```

### Gemini 2.5 Pro（バランス型）

**適用場面**:
- 一般的なビジネス文書作成
- 中程度の複雑さの分析タスク
- マルチモーダル処理
- 長いコンテキストの処理

**具体例**:
```
✅ プレゼン資料作成
✅ 企画書・提案書作成
✅ データ分析レポート
✅ 技術文書の翻訳・要約
```

### Gemini 2.5 Flash（高速処理）

**適用場面**:
- 迅速なレスポンスが必要なタスク
- 大量の簡単なタスク処理
- リアルタイム対話
- APIを通じた自動化

**具体例**:
```
✅ チャットボット応答
✅ 翻訳・要約の大量処理
✅ コードの簡単な修正
✅ メール下書き自動生成
```

## 専門機能の活用

### Deep Research - 深掘りリサーチ

!!! note "Deep Researchとは"
    ユーザーの質問を元に、インターネット上の信頼できる情報源（学術論文、ニュース記事、レポート等）から自動で情報収集し、構造化されたレポートを生成する機能です。

**使用手順**:
1. チャット画面右上の「🔍 Deep Research」を選択
2. 調査したいトピックを詳細に入力
3. Geminiが自動でリサーチプランを提案
4. 承認後、5-10分でレポート完成

**活用例**:
```
入力例: "2025年のリモートワーク市場動向と企業の対応策について調査して"

出力: 
- 市場規模の変化
- 主要企業の取り組み事例
- 技術的トレンド
- 今後の予測
- 推奨アクションプラン
```

**プロンプトのコツ**:
```
❌ 「AIについて調べて」（曖昧すぎ）
✅ 「生成AI市場の2025年Q4の成長率と主要企業の収益動向を、特に日本市場に焦点を当てて調査」
```

### Canvas - 統合編集環境

!!! tip "Canvasの利点"
    テキスト・コード・文書をリアルタイムで編集できる専用ワークスペース。ChatGPTのCanvasに近い機能で、Geminiとの対話をしながら直接編集作業ができます。

**主な用途**:

#### 1. 文書作成・編集
- **長文記事**の段階的な作成
- **構造化された報告書**
- **プレゼンテーション**の台本

**使い方**:
```
プロンプト例: "企業向けDXセミナーの企画書をCanvasで作成して"
→ Canvas上に企画書が表示
→ 「参加者数の想定を50名に変更して」で部分修正
```

#### 2. コード開発
- **Webアプリ**の作成
- **スクリプト**の編集・デバッグ
- **API連携**のコード

**実例**:
```javascript
// Canvasで生成されたTodoアプリのコード例
<!DOCTYPE html>
<html>
<head>
    <title>Todo Manager</title>
    <style>
        .todo-item { margin: 10px 0; }
        .completed { text-decoration: line-through; }
    </style>
</head>
<body>
    <h1>My Todo List</h1>
    <div id="todo-container"></div>
    <script>
        // Geminiが生成・編集するJavaScript
    </script>
</body>
</html>
```

#### 3. データ分析とビジュアライゼーション
- **グラフ**の生成
- **レポート**の構造化
- **ダッシュボード**作成

### Imagen - 画像生成

!!! warning "現在の制限"
    2026年2月時点で、Imagen（Geminiの画像生成機能）は一部地域で段階的に展開中です。

**特徴**:
- 自然言語による詳細な指示
- スタイル指定（写実、イラスト、抽象等）
- 高解像度出力

**活用例**:
```
プロンプト例: "モダンなオフィス空間で、多様な背景の人々がコラボレーションしている様子。明るい自然光、植物あり、写実的なスタイルで"
```

### Google検索統合

**リアルタイム情報取得**:
Geminiは自動でGoogle検索を活用し、最新情報を取得します。

**活用シーン**:
- 最新ニュースの分析
- 株価・相場情報
- 法改正・規制変更
- イベント・スケジュール確認

**プロンプト例**:
```
"今日の日経平均株価と前日比、主な変動要因を教えて"
→ 自動で最新情報を検索・分析して回答
```

### 長文コンテキスト活用

!!! info "100万トークンの威力"
    Geminiは最大100万トークン（約75万語）のコンテキストを処理可能。長大な文書の分析や、複数文書の比較分析が得意です。

**活用例**:

#### 1. 大容量PDF分析
```
"この200ページの技術仕様書から、セキュリティ関連の要件を全て抽出して表形式でまとめて"
```

#### 2. 複数文書の比較
```
"昨年と今年の年次報告書をアップロードするので、売上、利益、戦略の変化を分析して"
```

#### 3. 長編コンテンツの作成
```
"小説のプロット（3万字）をもとに、第1章（5千字）を詳細に執筆して"
```

## ファイルアップロード活用

### 対応フォーマット
- **PDF**: 学術論文、レポート、マニュアル
- **画像**: JPG, PNG, GIF, WebP
- **文書**: Word、Excel、PowerPoint
- **動画**: MP4, MOV, AVI（制限あり）
- **音声**: MP3, WAV（制限あり）

### 活用パターン

#### パターン1: 文書分析
```
PDFアップロード後:
"この契約書の主要なリスク条項を特定し、対応策を提案して"
```

#### パターン2: 画像からの情報抽出
```
会議ホワイトボードの写真をアップロード:
"この図を整理してミッションマップに構造化して"
```

#### パターン3: データ可視化
```
Excelファイルアップロード:
"売上データからトレンド分析グラフを作成し、見解をまとめて"
```

## モード選択の判断基準

| タスクの性質 | 推奨モード | 理由 |
|-------------|------------|------|
| **緊急性高** | Flash | 高速レスポンス |
| **精度重視** | Pro/3 Pro | 高品質出力 |
| **探索的** | Deep Research | 情報収集力 |
| **編集作業** | Canvas | 対話的編集 |
| **クリエイティブ** | 3 Pro + Canvas | 創造性+編集 |
| **日常業務** | 2.5 Pro | バランス良好 |

!!! tip "モード切り替えのコツ"
    一つのタスクで複数モードを組み合わせることも可能です。例えば：
    
    1. Deep Researchで情報収集
    2. 3 Proで分析・構造化
    3. Canvasで最終的な文書作成
