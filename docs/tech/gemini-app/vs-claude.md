### NotebookLM活用

```python
# NotebookLM + Gemini ワークフロー
sources = [
    "research_papers/",
    "meeting_notes/", 
    "project_docs/"
]

# 1. NotebookLMで知識ベース構築
notebook = create_notebook(sources)

# 2. Geminiで100万トークン分析
analysis = gemini.analyze_with_context(
    query="競合他社の戦略パターンを抽出",
    context_size="1M_tokens"
)

# 3. 統合レポート生成
report = notebook.generate_report(analysis)
```
