# Google Workspace連携

GeminiとGoogle Workspaceの連携により、Gmail、Docs、Sheets、Meet などの日常業務ツールでAI支援を受けることができます。業務効率化と生産性向上のための実践的な活用法を解説します。

## Workspace連携の概要

!!! info "Gemini for Workspace とは"
    Google WorkspaceアプリにGeminiが統合され、各アプリ内で直接AI支援を受けられる機能です。Google AI Pro/Ultra プランで利用可能です。

**連携可能アプリ**:
- **Gmail**: メール作成・要約・検索
- **Google Docs**: 文書作成・編集・校正
- **Google Sheets**: データ分析・関数生成・レポート作成
- **Google Slides**: プレゼン作成・デザイン提案
- **Google Meet**: 議事録・要約・アクション抽出
- **Google Drive**: ファイル検索・整理・要約
- **Google Chat**: チーム コミュニケーション支援

### 前提条件

**必要なプラン**:
- Google AI Pro（月額2,900円）以上
- または Google Workspace with Gemini Business/Enterprise

**設定手順**:
1. Google アカウントでGemini Advanced（AI Pro）に加入
2. Workspace Apps で「Gemini Apps を有効にする」
3. 各アプリで Gemini 機能を有効化

## Gmail × Gemini

### メール作成支援

**基本的な使い方**:
1. Gmailで「作成」をクリック
2. 右下の「✨ Help me write」を選択
3. 作成したいメールの概要を入力

**実用例**:

```
【レポートの要点】

来週にでもお時間をいただき、詳細をご説明させていただければと思います。

今後ともどうぞよろしくお願いいたします。
```

### 文書校正・改善

**Refine 機能**:
既存の文書を選択して、以下の改善ができます：

```
原文:
"この商品は売上が良くて、お客さんからの評価も高いです。なので、さらに販売を拡大していくべきだと思います。"

改善後:
"本商品は売上実績が好調で、顧客満足度も高い評価を獲得しています。これらの成果を踏まえ、販売拡大戦略の検討を推奨します。"

改善ポイント:
- より専門的で適切な表現に変更
- 論理的な構成に整理
- ビジネス文書として適切なトーン
```

### 文書構成の改善

```
プロンプト例:
"この報告書の構成を改善し、読みやすくして"

Before:
- 情報が雑然と並んでいる
- 見出しが不明確
- 結論が分かりにくい

After:
1. エグゼクティブサマリー（要点を最初に）
2. 背景・目的（なぜこの調査を行ったか）
3. 方法・範囲（どのように調査したか）  
4. 結果・発見事項（何が分かったか）
5. 結論・推奨事項（何をすべきか）
6. 付録・詳細データ
```

## Google Sheets × Gemini

### データ分析支援

**Help me organize 機能**:
1. データ範囲を選択
2. 右クリックで「Help me organize」を選択
3. 分析したい内容を指示

**実用例**:

#### 売上データ分析
```
データ: 月別売上実績（12ヶ月分）

プロンプト: "月別売上の傾向分析とグラフ作成"

生成される結果:
1. 売上トレンドのライングラフ
2. 前年同月比較棒グラフ
3. 四半期別集計表
4. 主要な傾向の分析コメント
   - "7-9月の売上が特に好調"
   - "12月は前年比120%の成長"
   - "来期Q1の予測は..."
```

#### 顧客分析ダッシュボード
```
データ: 顧客属性・購買履歴

プロンプト: "顧客セグメント分析のダッシュボード作成"

生成機能:
- 年代別顧客数の円グラフ
- 購買金額別ヒストグラム
- 商品カテゴリ別売上
- LTV（Life Time Value）計算
- リピート率分析
```

### 関数・数式生成

**Help me calculate 機能**:

```
ユーザー入力: "売上データから前年同月比の成長率を計算したい"

AI生成式:
=((B2-C2)/C2)*100

説明:
- B2: 今年の売上
- C2: 昨年同月の売上
- 結果: パーセンテージでの成長率
```

**複雑な条件分岐**:
```
要件: "売上目標達成率に応じてボーナス率を計算"
- 120%以上: 15%
- 110-119%: 10%  
- 100-109%: 5%
- 100%未満: 0%

AI生成式:
=IF(B2>=1.2,0.15,IF(B2>=1.1,0.1,IF(B2>=1,0.05,0)))
```

### レポート自動生成

```
プロンプト: "月次販売レポートを自動生成するテンプレート"

2. データシート
```

## 実務での活用ワークフロー

### ワークフロー例1: 月次レポート作成

```
1. Gmail で顧客からの報告依頼を受信
   → Geminiで返信メール自動生成（納期確認）

2. Sheets で売上データを分析
   → 「月次分析レポート作成」プロンプトで自動グラフ化

3. Docs でレポート本文を作成
   → Sheets のデータを参照しつつ、分析コメント生成

4. Slides でプレゼン資料を作成
   → Docs の内容をベースに、視覚的な資料に変換

5. Meet で顧客への報告会
   → 自動議事録で次回アクションを明確化

6. Gmail で議事録とレポートを送信
   → 礼状とフォローアップメールを自動生成
```

### ワークフロー例2: 企画書作成プロセス

```
段階1: リサーチ（Gemini Deep Research）
- 市場調査
- 競合分析
- トレンド調査

段階2: データ整理（Sheets + Gemini）
- 調査結果の数値化
- 比較表の作成
- 予算試算

段階3: 企画書執筆（Docs + Gemini）
- 構成案の自動生成

段階4: プレゼン準備（Slides + Gemini）
- スライド構成の最適化
- 視覚効果の提案

段階5: プレゼン実施（Meet + Gemini）
```

## セキュリティ・プライバシー

### データの取り扱い

!!! warning "機密情報の注意"
    - Geminiに送信されたデータはGoogleサーバーで処理
    - 機密性の高い情報（個人情報、企業秘密等）の入力は避ける
    - 必要に応じてデータの匿名化・マスキング処理を実施

**推奨セキュリティ対策**:
1. **機密度の分類**: 公開可能な情報のみAI利用
2. **アクセス制御**: 必要最小限のメンバーのみ利用
3. **ログ監視**: Gemini利用履歴の定期確認
4. **ガイドライン策定**: 社内利用ルールの明文化
