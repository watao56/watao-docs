# Gems - カスタムGemini

GemsはGeminiをカスタマイズして、特定の目的やタスクに特化したAIアシスタントを作成できる機能です。ChatGPTのカスタムGPTsに相当する機能で、個人やチームの作業効率を大幅に向上させます。

## Gemsとは

!!! info "Gemsの概要"
    Gemsは、特定の役割・タスク・スタイルに合わせてカスタマイズされたGeminiです。事前に設定した指示に基づいて、一貫した応答を提供します。

**主な特徴**:
- **専門性**: 特定分野に特化した知識と応答
- **一貫性**: 定義されたペルソナやスタイルを維持
- **効率性**: 繰り返し使用するプロンプトの自動化
- **共有性**: 作成したGemsをチーム内で共有可能

### ChatGPTカスタムGPTsとの比較

| 機能 | Gemini Gems | ChatGPT カスタムGPTs |
|------|-------------|---------------------|
| **基盤モデル** | Gemini 2.5/3 Pro/Flash | GPT-4/4o |
| **Google統合** | ✅ 検索・Workspace連携 | ❌ |
| **マルチモーダル** | ✅ 画像・動画・PDF対応 | ✅ 画像のみ |
| **共有方法** | リンク共有 | 公開・プライベート |
| **プリメイドGems** | ✅ 公式テンプレート豊富 | ✅ GPTストア |
| **無料利用** | ✅ 制限付きで可能 | ❌ Plus必須 |
| **API連携** | 開発中 | ✅ 利用可能 |

## Gems作成ガイド

### 基本作成手順

1. **Geminiアプリにアクセス**
2. **Gem Manager を開く**: 右上の「💎 Gems」をクリック
3. **新規作成**: 「Create New Gem」を選択
4. **基本設定を入力**:
   - **名前**: 分かりやすい名前
   - **説明**: Gemの目的と特徴
   - **アイコン**: 視覚的識別用
5. **指示を作成**: 詳細なカスタム指示を入力
6. **プレビューテスト**: 実際に動作確認
7. **保存・共有**: 完成したGemを保存

### 効果的なカスタム指示の作成

#### 4つの重要要素

**1. ペルソナ（役割定義）**
```
✅ 良い例:
"あなたは10年以上の経験を持つUX/UIデザイナーです。ユーザー中心設計の原則に基づき、アクセシビリティとユーザビリティの両方を重視します。"

❌ 悪い例:
"デザイナーとして答えてください。"
```

**2. タスク（具体的な作業内容）**
```
✅ 良い例:
"Webサイトの改善提案を行う際は、以下の順序で分析してください：
1. 現状のユーザビリティ評価
2. 課題の優先度付け
3. 具体的な改善案（3つ以上）
4. 実装難易度とビジネスインパクトの評価
5. プロトタイプのワイヤーフレーム提案"

❌ 悪い例:
"Webサイトを分析してください。"
```

**3. 出力形式**
```
✅ 良い例:
"回答は以下の形式で構造化してください：
- エグゼクティブサマリー（3行以内）
- 詳細分析（見出し付き）
- 推奨アクション（期限・担当者付き）
- 参考資料・追加調査項目"
```

**4. 制約・ガイドライン**
```
✅ 良い例:
"制約事項：
- 予算は月額50万円以内
- 実装期間は3ヶ月以内
- 既存システムとの互換性を維持
- セキュリティ要件（金融業界準拠）を満たす"
```

### 実践例：営業支援Gem

```
Gemの名前: 営業戦略アドバイザー

ペルソナ:
あなたは15年の営業経験を持つ営業本部長です。B2B営業に精通し、顧客の真のニーズを的確に把握し、長期的な関係構築を重視します。データ分析に基づく戦略立案と、現場での実践的なソリューション提案が得意です。

タスク:
営業活動に関する相談に対して、以下の観点から分析・提案を行ってください：

1. 顧客分析（業界・規模・課題・意思決定プロセス）
2. 競合状況の把握
3. 提案内容の最適化
4. アプローチ戦略の立案
5. 成果測定指標の設定

出力形式:
- 現状分析（簡潔に）
- 戦略的提案（3つの選択肢）
- 具体的アクション（時系列）
- リスク要因と対策
- 成功確率の評価

制約:
- 提案は実現可能性を重視
- 倫理的な営業手法のみ推奨
- 長期的な顧客満足を最優先
- ROI の明確な根拠を示す
```

### プリメイドGemのカスタマイズ

!!! tip "プリメイドGemを活用"
    Google提供の公式GemsやコミュニティGemsをベースに、自社・個人の要件に合わせてカスタマイズできます。

**カスタマイズ手順**:
1. 気に入ったプリメイドGemを選択
2. 「Copy to create custom Gem」をクリック
3. 指示内容を自社向けに修正
4. 業界特有の知識・制約を追加
5. 出力形式を社内標準に合わせて調整

**カスタマイズ例**:
```
元: 一般的な「Email Assistant」
↓
カスタマイズ後: 「不動産業界向けメール営業アシスタント」

追加要素:
- 不動産業界の専門用語
- 法的制約（宅建業法等）
- 一般的な顧客の懸念点
- 業界標準のメール署名フォーマット
```

## 設定のコツ

### 1. 段階的な改善

```
初回作成 → テスト → フィードバック収集 → 指示の調整 → 再テスト
```

**改善ポイント**:
- 期待と異なる回答が出た部分を特定
- より具体的な指示に修正
- 例示を追加
- 制約条件を明確化

### 2. 用途別の特化

```
❌ 汎用的すぎる:
"ビジネス全般のアドバイザー"

✅ 特化型:
"SaaS企業のカスタマーサクセス戦略アドバイザー"
```

### 3. チーム共有の最適化

**共有前チェックリスト**:
- [ ] 会社固有の機密情報が含まれていないか
- [ ] チームメンバーが理解しやすい説明があるか
- [ ] 想定外の使用方法での動作確認済みか
- [ ] アップデート時の影響範囲を把握しているか

## Gemsの管理と運用

### バージョン管理

```
Gem名前付けルール例:
- マーケティング分析_v2.1
- 顧客対応_2026年2月版
- 財務レポート_四半期用
```

### 効果測定

**KPI例**:
- 使用頻度（週・月次）
- 満足度（5段階評価）
- 時間短縮効果（タスクあたり）
- 品質向上度（アウトプットの評価）

### セキュリティ考慮事項

!!! warning "機密情報の取り扱い"
    - 会社の機密情報をGemの指示に含めない
    - 個人情報を含むプロンプト例は使用しない
    - 共有範囲を適切に設定
    - 定期的な見直しと削除
