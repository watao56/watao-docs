# 効果的なプロンプト設計

Geminiで最高の結果を得るには、プロンプトの書き方が重要です。Gemini特有の機能を活かし、マルチモーダルな入力を効果的に活用するテクニックを解説します。

## Gemini特有のプロンプトテクニック

### 1. Google検索グラウンディングの活用

!!! tip "リアルタイム情報取得"
    Geminiは自動でGoogle検索を利用します。最新情報が必要な場合は明示的に指示しましょう。

```
❌ 悪い例:
"日本の現在の首相について教えて"

✅ 良い例:
"2026年2月現在の日本の首相と最新の政策動向を、直近1ヶ月のニュースを含めて教えて"
```

### 2. 段階的思考の促進

Geminiは思考プロセスを明示することで、より正確な回答を生成します。

```
✅ 推奨フォーマット:
"以下の手順で分析してください：
1. 現状分析（データに基づく）
2. 課題の特定（優先度付き）
3. 解決策の検討（3つの選択肢）
4. 推奨アクション（具体的ステップ）

対象：新商品のマーケティング戦略"
```

### 3. コンテキスト長を活かした指示

100万トークンの大容量コンテキストを活用しましょう。

```
✅ 長文コンテキスト活用:
"以下3つの文書を読み込んで総合分析してください：
- 競合分析レポート（3万字）
- 市場調査データ（2万字） 
- 顧客インタビュー（1万字）

分析観点：
1. 共通トレンド
2. 矛盾点・課題
3. 戦略への示唆"
```

## マルチモーダルプロンプト

### 画像 + テキスト

**基本パターン**:
```
[画像をアップロード]
"この画像について以下の観点で分析してください：
1. 視覚的要素の特徴
2. 改善点の提案
3. 活用方法のアイデア"
```

**デザイン分析例**:
```
[ウェブサイトのスクリーンショット]
"このWebサイトのUX/UI を以下の観点で評価してください：
- ナビゲーションの使いやすさ
- 視覚的階層の明確さ
- モバイル対応度
- アクセシビリティ
- 改善提案（具体的な修正箇所を指摘）"
```

**データ可視化例**:
```
[グラフの画像]
"このグラフから以下を読み取ってください：
1. 主要トレンド（時系列変化）
2. 異常値・特異点
3. ビジネスインパクト
4. 次期予測（根拠付き）"
```

### 文書 + 画像の組み合わせ

```
[PDFレポート + 関連画像を複数アップロード]
"この市場調査レポートと添付の図表を総合して：
1. 市場機会の特定
2. 競合ポジショニング分析
3. 戦略提案（3つのシナリオ）
4. リスク要因と対策"
```

### 動画・音声入力

!!! note "動画・音声対応"
    2026年現在、Geminiは動画・音声ファイルの処理にも対応しています（一部制限あり）。

```
[会議録画をアップロード]
"この会議動画から以下を抽出してください：
1. 主要な決定事項
2. アクションアイテム（担当者・期限付き）
3. 未解決の課題
4. 次回議題の提案"
```

## タスク別プロンプト例

### 1. ビジネス文書作成

**企画書**:
```
"以下の要件で企画書を作成してください：

背景：
- 市場環境：[具体的な状況]
- 課題：[解決したい問題]
- 目標：[達成したい成果]

構成：
1. エグゼクティブサマリー
2. 市場分析
3. 提案内容
4. 実行計画
5. リスク評価
6. ROI試算

制約：
- A4で10ページ以内
- 役員向けプレゼンを想定
- データは表・グラフで可視化"
```

**レポート作成**:
```
"月次業績レポートのテンプレートを作成してください：

含めるべき要素：
- KPI ダッシュボード（5つの主要指標）
- トレンド分析（前月・前年比較）
- 課題と対策
- 来月の重点項目

フォーマット：
- 視覚的にわかりやすく
- 経営陣が5分で把握できる構成
- アクションが明確"
```

### 2. 技術・開発支援

**コードレビュー**:
```python
# [コードをペースト]
"""
このPythonコードを以下の観点でレビューしてください：

1. セキュリティ脆弱性
2. パフォーマンス最適化
3. 可読性・保守性
4. エラーハンドリング
5. ベストプラクティス準拠

改善提案は具体的なコード例と共に示してください。
"""
```

**システム設計**:
```
"以下の要件でシステム設計を提案してください：

要件：
- ユーザー数：10万人規模
- データ量：1TB/月の増加
- 可用性：99.9%以上
- セキュリティ：金融レベル

検討事項：
1. アーキテクチャパターン
2. 技術スタック
3. インフラ構成
4. セキュリティ対策
5. 運用・監視方法
6. スケーラビリティ対応"
```

### 3. 学習・教育支援

**学習プラン**:
```
"以下のテーマの学習プランを作成してください：

テーマ：機械学習入門（3ヶ月間）
対象レベル：プログラミング経験あり、ML初心者
学習時間：週10時間

含めるべき要素：
1. 週次学習目標
2. 推奨教材（書籍・オンラインコース）
3. 実践プロジェクト
4. 理解度チェック方法
5. 次段階への準備"
```

**クイズ生成**:
```
[学習資料をアップロード]
"この資料から理解度確認用のクイズを作成してください：

形式：
- 選択問題：10問
- 記述問題：5問
- 応用問題：3問

レベル：中級者向け
解説付きで、間違いやすいポイントも含めてください。"
```

## Google検索グラウンディングの活用

### 最新情報の取得

```
"以下について最新の動向を調査してください：
- 検索期間：過去3ヶ月
- 情報源：信頼できるメディア・公式発表を優先
- 分析観点：市場への影響度

テーマ：生成AI市場の規制動向"
```

### 事実確認と検証

```
"以下の主張について事実確認を行ってください：

検証対象：「2025年のテレワーク普及率は70%を超えた」

検証方法：
1. 公式統計データの確認
2. 複数の調査機関データの比較
3. 地域・業界別の詳細分析
4. 信頼性の評価

結果は根拠とともに示してください。"
```

## ChatGPTからの移行のコツ

### 1. プロンプト変換パターン

| ChatGPT | Gemini |
|---------|--------|
| "Act as..." | "あなたは...の専門家として" |
| "Step by step" | "以下の手順で段階的に" |
| "Think aloud" | "思考プロセスを明示しながら" |
| "In the style of..." | "...のスタイルで（具体的な説明付き）" |

### 2. 情報源の指定

```
ChatGPT（限界あり）：
"最新の情報で..."

Gemini（Google検索活用）：
"最新のGoogle検索結果を含めて、2026年2月現在の..."
```

### 3. 多言語対応

Geminiは日本語での複雑な指示にも自然に対応します：

```
✅ Gemini向け自然な日本語：
"この企画について、ステークホルダーの立場から見たメリット・デメリットを整理し、懸念点への対策も含めて包括的に分析してもらえますか？"
```

## 長文コンテキストの活用法

### 複数文書の統合分析

```
[複数のPDFをアップロード]
"これらの文書を統合して以下を作成してください：

1. 共通テーマの抽出
2. 矛盾する見解の特定
3. 包括的な結論
4. さらなる調査が必要な領域

出力形式：エグゼクティブサマリー（2ページ）"
```

### 長文コンテンツの構造化

```
"この15,000字のブログ記事を以下の形式で再構成してください：

目的：読みやすさの向上
構成：
- 導入部（3分で読める）
- 主要ポイント（5つのセクション）
- 実践的なアクション（チェックリスト形式）
- まとめ（キーポイントの箇条書き）

各セクションに適切な見出しを付けてください。"
```

## プロンプト最適化のチェックリスト

!!! note "プロンプト品質チェック"
    ✅ 目的が明確に定義されている  
    ✅ 期待する出力形式が指定されている  
    ✅ 制約条件が明記されている  
    ✅ 例示や具体的な基準がある  
    ✅ 文脈情報が十分に提供されている  
    ✅ マルチモーダル要素が適切に活用されている
